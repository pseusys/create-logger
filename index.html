Welcome to create-logger!

Create-logger is a web tool to create simple yet colorful console logging module for your own app!

STYLE tab

Style tab is a WYSIWYG text editor with limited functionality. The only active line is marked with white shadow around the line number.

Here you can set front and back color of selected text and also apply properties like underscored or blinking to it.

You can use presets to create and store formatting styles.

It is also possible to assign variable name and type to specially-formatted text parts.

Visit project wiki for more info!
GENERAL tab

General tab contains basic web tool settings.

Here you can set front and back color of terminal window.

Set important flags and pass arguments to selected code generator.

Select web tool language and code generation language.

Visit project wiki for more info!
PREVIEW tab

Preview tab contains generated ASCII escape sequences for each of the formatted lines.

Visit project wiki for more info!
CODE tab

Code tab contains source code of logging module generated by selected code generation plugin.

Visit project wiki for more info!
